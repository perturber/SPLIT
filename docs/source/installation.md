# Installation

Currently, SPLIT is available by building from the source. 

```bash
git clone [https://github.com/perturber/SPLIT.git](https://github.com/perturber/SPLIT.git)
cd SPLIT
pip install .
```

To install with GPU support for `FastEMRIWaveforms` and `Eryn`:
```bash
pip install .[gpu]
```

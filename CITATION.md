# Citing SPLIT

If you use SPLIT in your research, please cite it via its archived
Zenodo record: [10.5281/zenodo.20290209](https://doi.org/10.5281/zenodo.20290209).

## BibTeX

```bibtex
@software{kejriwal_2026_20290209,
  author       = {Kejriwal, Shubham},
  title        = {SPLIT - Sliced Posteriors for Long-Inspiral
                   Trajectories
                  },
  month        = may,
  year         = 2026,
  publisher    = {Zenodo},
  version      = {v0.0.1alpha},
  doi          = {10.5281/zenodo.20290209},
  url          = {https://doi.org/10.5281/zenodo.20290209},
}
```

The Zenodo DOI above is a *concept DOI* — it always resolves to the
latest archived release. To cite a specific version, use the
version-specific DOI shown on the corresponding Zenodo record page.
